## Project Overview

3DVLMReasoning is a two-stage framework for 3D scene understanding with evidence-seeking VLM agents. Stage 1 performs query-driven keyframe retrieval from ConceptGraph 3D scene graphs, and Stage 2 uses a ReAct-style VLM agent (GPT-5.2 via Azure) for agentic reasoning over the retrieved visual evidence. The project evaluates against the OpenEQA benchmark using 89 locally prepared ScanNet scenes.

A comprehensive 100-case evaluation (20 scenes x 5 questions) was completed on 2026-03-27, revealing critical architectural bottlenecks in the Stage 1 retrieval pipeline — primarily the majority-vote information loss that drops ~65% of detected object classes from the scene vocabulary. An optimization plan was created and implementation is in progress on the current branch.
