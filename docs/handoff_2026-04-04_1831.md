## Project Overview

3DVLMReasoning is a two-stage framework for 3D scene understanding with evidence-seeking VLM agents, evaluated on the OpenEQA benchmark. Stage 1 retrieves keyframes from 3D ConceptGraph scene graphs; Stage 2 uses a VLM (GPT-5.4) with tool callbacks to reason over retrieved evidence.
