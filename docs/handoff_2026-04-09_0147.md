## Project Overview

3DVLMReasoning is a two-stage framework for 3D scene QA on the OpenEQA benchmark (CVPR 2024). Stage 1 retrieves keyframes from ConceptGraph 3D scene graphs using LLM-enriched object labels. Stage 2 uses GPT-5.4 with evidence-seeking tools to reason over visual evidence. The system achieves **MNAS 73.1** on 1050 ScanNet questions (v14), surpassing GPT-4V (51.3) by +21.8.

The current task is **visualizing the full OpenEQA pipeline** ‚Äî creating interactive HTML visualizations that show the complete reasoning trace for each question.
